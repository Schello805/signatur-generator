Anwendungshinweise (Gmail/Outlook)
  • Gmail: Einstellungen → „Alle Einstellungen aufrufen“ → „Allgemein“ → „Signatur“ → im Editor einfügen (Rich‑Text).
  • Outlook (Desktop): Datei → Optionen → E‑Mail → Signaturen → neue Signatur → einfügen (Rich‑Text).
  • Wenn Bilder nicht angezeigt werden: nutze eine öffentliche HTTPS‑Bild‑URL statt Upload/Base64.
  • Wenn „Kopieren“ nicht funktioniert: öffne die App über localhost (z. B. python3 -m http.server) statt direkt per Datei‑Doppelklick.
