# Code of Conduct

Kurzfassung: Sei respektvoll, konstruktiv und hilfsbereit.

Dieses Projekt orientiert sich inhaltlich am „Contributor Covenant“, ohne den Text vollständig zu kopieren.

- Keine Beleidigungen, Diskriminierung oder Belästigung.
- Kritik am Code ist ok – Kritik an Personen nicht.
- Konflikte: bitte deeskalieren und sachlich bleiben.
