# Contributing

Danke, dass du helfen möchtest.

## Grundsätze

- Bitte keine Scraper/Quellen hinzufügen, die klar gegen Nutzungsbedingungen verstoßen oder Rate-Limits umgehen.
- Kleine, fokussierte PRs sind am besten.
- Keine großen Umformatierungen ohne Absprache.

## Lokales Setup

- Node Server: `node server/server.js`
- Webapp: `http://127.0.0.1:8000/pelletpreise/`

## Qualitätschecks

- Syntaxcheck: `npm run verify`

## Code Style

- JavaScript (ESM), ohne Build-Step.
- Keine Inline-Kommentare als „Doku“ – lieber klare Funktionen und gute Dateinamen.
