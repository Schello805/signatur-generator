Datenverarbeitung in der Webapp

Was passiert mit deinen Eingaben?

  • Die Webapp läuft vollständig im Browser. Die von dir eingegebenen Daten werden nicht an einen Server übertragen.
  • Optional kannst du „Im Browser speichern (LocalStorage)“ aktivieren. Dann werden deine Eingaben ausschließlich lokal auf deinem Gerät im Browser gespeichert und können jederzeit über „Zurücksetzen“ bzw. durch Deaktivieren der Speicherung entfernt werden.
  • Beim Export („Kopieren“, „Download .html“, „Download vCard“) werden Daten nur auf deinem Gerät in die Zwischenablage bzw. als Datei verarbeitet.

Hinweis: Wenn du ein Bild als Data‑URL einfügst, kann es sein, dass E‑Mail‑Programme das blockieren. Das ist eine Kompatibilitätsfrage, keine Übertragung an einen externen Dienst.
