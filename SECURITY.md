# Security Policy

## Supported Versions

Dieses Projekt ist eine lokale Webapp ohne automatische Updates. Sicherheitsfixes erfolgen nach Bedarf.

## Reporting

Bitte melde Sicherheitslücken verantwortungsvoll:

- Keine Details in öffentlichen Issues posten.
- Stattdessen einen privaten Kanal nutzen (z. B. GitHub Security Advisory, falls aktiviert).

## Scope

- XSS/Injection im UI
- Path Traversal / Directory Traversal im lokalen Server
- Unsichere Default-Konfigurationen
